# PIN MAP
 ESP32 Pin Mapping Table 

 Blue LED   	D2	 Output	
 Red LED	    D4	 Output	
 Green LED  	D5	 Output	
 Button 1	    D35  Input	
 Button 2	    D32  Input	
 Buzzer	      D25  Output 
 OLED SDA	    D21  I²C Data	
 OLED SCL	    D22  I²C Clock	
 OLED VCC	    3V3  Power	
 OLED GND	    GND  Ground
